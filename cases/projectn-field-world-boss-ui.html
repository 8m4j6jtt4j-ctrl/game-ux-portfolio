필드·월드 보스 UI 개편

필드/월드 보스 콘텐츠의 현황판, 보상 팝업, 권장 전투력, 채널 이동 편의 등을 정리한 콘텐츠 UX 개선 사례입니다.

Role
보스 콘텐츠 UI 개편, 현황/보상/이동 편의 UX 정리
Period
2021.01-2021.11

Planning Context

어떤 기획이었나

  • 필드/월드 보스는 출현 상태, 권장 전투력, 보상 확인, 채널/파티 이동이 빠르게 연결되어야 하는 콘텐츠입니다.
  • 정보가 흩어지면 유저가 보스 상태와 참여 가능 여부를 판단하기 어렵습니다.

Process

어떻게 작업했나

  • 현황판, 처치 결과, 보상 팝업, 권장 전투력 표시, 채널 이동 편의 기능을 분리해 정리했습니다.
  • 참여 전 확인해야 하는 정보와 전투 후 확인해야 하는 결과 정보를 나눠 화면 역할을 정의했습니다.
  • 라이브 개선 문서와 시안을 함께 확인해 콘텐츠 진입/결과 흐름을 맞췄습니다.

Deliverables

결과물

  • M6_필드_월드보스 화면 개선_스크립트.pptx/pdf
  • QA5_필드 월드 보스 UI 개편_스크립트.pptx/pdf
  • 필드/월드보스 보상 팝업 개선 문서

Result

도출한 결과

보스 콘텐츠의 참여 판단, 전투 결과, 보상 확인, 이동 편의를 하나의 콘텐츠 UX 흐름으로 정리했습니다.
