테트로 퍼즐 UX 개선

테트로 퍼즐의 장착, 가방, 메인 화면과 성장 인지를 개선한 시스템 UX 사례입니다.

Role
테트로 퍼즐 UX 개선, 장착/가방/성장 상태 흐름 정리
Period
2020.09-2020.12

Planning Context

어떤 기획이었나

  • 퍼즐형 성장 시스템은 보유, 장착, 성장 가능 상태가 분리되어 있어 유저가 다음 행동을 판단하기 어려울 수 있습니다.
  • 가방과 메인 화면의 정보 구조가 명확하지 않으면 성장 동기와 조작 흐름이 약해집니다.

Process

어떻게 작업했나

  • 테트로 퍼즐 메인, 가방, 장착 상태, 성장 가능 상태를 화면 단위로 나눠 검토했습니다.
  • 유저가 보유 퍼즐과 장착 결과를 비교하는 흐름을 기준으로 정보 우선순위를 정리했습니다.
  • 스크립트 문서와 시안을 함께 보며 상태 표기와 버튼 동작을 맞췄습니다.

Deliverables

결과물

  • M6_테트로 퍼즐 UX 개선_스크립트.pptx/pdf
  • 테트로 퍼즐 메인/가방 UI 시안
  • 장착 및 성장 상태별 UX 스크립트

Result

도출한 결과

퍼즐 성장 시스템의 보유, 장착, 성장 가능 상태를 연결해 유저가 다음 행동을 이해하기 쉬운 구조로 정리했습니다.
