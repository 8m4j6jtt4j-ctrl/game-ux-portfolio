초상화 시스템 UX

초상화 시스템의 조합, 특수 연결, 성장/수집 구조를 정리한 컬렉션 시스템 UX 사례입니다.

Role
초상화 시스템 UX 스크립트, 조합/연결/수집 흐름 정리
Period
2026.05

Planning Context

어떤 기획이었나

  • 컬렉션형 시스템은 수집, 조합, 연결, 성장 결과가 복합적으로 얽혀 있어 정보 구조가 중요합니다.
  • 새 시스템의 첫 진입에서 유저가 목표와 보상, 조작 방식을 빠르게 이해해야 합니다.

Process

어떻게 작업했나

  • 초상화 메인, 조합 UI, 특수 연결, 결과 확인 흐름을 화면 단위로 나눠 정리했습니다.
  • 수집 목표와 조합 가능 상태, 연결 효과를 유저가 판단하는 순서로 정보 우선순위를 잡았습니다.
  • UX 스크립트와 시안을 함께 보며 상태값과 안내 문구를 정리했습니다.

Deliverables

결과물

  • 7UD_초상화 시스템_스크립트.pptx
  • 마물의초상화_UX스크립트_v1.0.pptx
  • 초상화/조합/특수 연결 UI 시안

Result

도출한 결과

컬렉션 시스템의 수집, 조합, 연결 흐름을 하나의 성장 UX로 묶어 신규 시스템 이해도를 높이는 방향으로 정리했습니다.
