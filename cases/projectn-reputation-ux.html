명성 UX 개선

명성 시스템의 세력 현황, NPC 상세, 퀘스트 안내와 잠금 조건을 유저가 이해할 수 있도록 정리한 시스템 UX 사례입니다.

Role
명성 시스템 UX 개선, 퀘스트/상태 안내 흐름 정리
Period
2020.12-2021.05

Planning Context

어떤 기획이었나

  • 명성 시스템은 세력, NPC, 퀘스트, 잠금 조건, 추천 동선이 함께 얽혀 있어 유저가 다음 행동을 놓치기 쉽습니다.
  • 퀘스트와 세력 상태를 화면별로 나눠 보여주면 전체 진행 구조와 보상/해금 목적이 약해질 수 있습니다.

Process

어떻게 작업했나

  • 세력 현황, NPC 상세, 퀘스트 목록, 잠금 안내, 배지/알림 흐름을 화면 단위로 분리했습니다.
  • 유저가 현재 가능한 퀘스트와 잠긴 퀘스트를 구분할 수 있도록 상태 표시와 안내 문구를 정리했습니다.
  • 시안과 UX 스크립트가 같은 조건과 문구를 가리키도록 확인했습니다.

Deliverables

결과물

  • M6_명성 UX 개선_스크립트.pptx/pdf
  • 명성 UI 개선 시안
  • 명성 퀘스트 안내/잠금 표시 관련 스크립트

Result

도출한 결과

명성 시스템의 진행 조건과 세력/NPC/퀘스트 정보를 연결해 유저가 다음 목표를 파악하기 쉬운 구조로 정리했습니다.
